# My Emacs configuration

## Highlights

- Based on vanilla Emacs.
- [Elpaca][elpaca] for declarative package management.
- Modern and performant completions thanks to the MOVEC stack
  ([Marginalia][marginalia], [Orderless][orderless],
  [Vertico][vertico], [Embark][embark], [Consult][consult]).
- [Corfu][corfu] for in-buffer completion popups.

## Installation

Clone this repository to `~/.emacs.d`.
